## Universal Console

Use your favorite browser's console from NodeJS, React Native or Electron

## Install

```sh
  yarn add -D universal-console
```

## Usage

```typescript
import uc from "universal-console";

uc.log(1, 2, { foo: 'bar' });

uc.warn(1, 2, { foo: 'bar' });

uc.error(1, 2, { foo: 'bar' });

```

And check your logs on http://localhost:7331
